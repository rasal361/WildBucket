# 🏕 WildBucket — Adventure Bucket List Tracker

A beautifully crafted, fully offline adventure bucket list tracker built with pure HTML, CSS and JavaScript.

## 🚀 How to Run

Simply open `index.html` in any modern web browser. No installation or server required!

## ✨ Features

### Core
- **Add Adventures** — name, country, category, priority, target year, and notes
- **Mark Complete** — tick off adventures with a satisfying confetti celebration 🎉
- **Edit & Delete** — full CRUD management for every item
- **local Storage** — all data persists automatically between sessions

### Views
- **Dashboard** — stats overview, progress bar, recent adventures, category breakdown
- **My Adventures** — full list with search, filter by category/priority, and sort options
- **Completed** — showcase of conquered adventures
- **Badges** — achievement system unlocked by completing milestones

### Smart Features
- ✦ **Inspire Me** — one-click random adventure suggestion from 16 curated ideas
- 🏅 **12 Badges** — earn achievements like "Globetrotter", "Mountain Trekker", "Sky High"
- 📊 **Live Stats** — adventures count, completion rate, countries, categories
- 🔍 **Search & Filter** — find adventures by name, country, category, or priority
- 📱 **Fully Responsive** — works great on mobile, tablet, and desktop
